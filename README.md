# ADMIN-CHECKER-PLUS
ADMIN CHECKER PLUS - панель управления Windows, объединяющая все системные утилиты в одном окне. Показывает имя компьютера, время и статус администратора. Позволяет мгновенно запускать CMD, PowerShell, Проводник, Диспетчер задач и другие инструменты для управления системой без поиска в меню.
